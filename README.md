**Movie Recommendation System 🎬**

This project leverages machine learning to recommend movies based on the popular 5000 TMDB Movies and Credits Dataset. Built and deployed using Streamlit, the app provides interactive recommendations to users in a straightforward and accessible format.

**Features**
 Dataset: Utilizes TMDB's 5000 Movies and Credits dataset for a comprehensive movie recommendation system. 
 Tech Stack: Developed using Streamlit for a seamless, interactive user experience, both in building and deployment.
 Deployment: The deployed app is available in the Deployment folder with a link to access the live application.


**Usage**
 Clone the Repository: git clone https://github.com/yourusername/Movie-Recommendation-System.git
 Navigate to Project: Open the project in PyCharm for easy access.
 Run Streamlit App: Execute the app locally with streamlit run app.py, or explore the deployed version.
